## 네이밍 규칙

### 예시
- PR : `[week 1] 자료구조 1 - 3문제`
- 문제 : `🔑 B1111 / 탑 쌓기 /실버4`
- 파일명 :   `B1111.py`

### 소스코드 파일명
```
[문제출처] :  B / P / L /...  + 문제번호 
```

### Commit

```
 🔒 시도했지만 풀지 못한 문제 or 답안을 참고한 문제
 🔑 해결한 문제

scope : B / P / L / ... {문제 번호}
백준 / 프로그래머스 / 리트코드 / ..

문제 : {문제이름}

문제 랭크 : 브론즈3, 실버2 ...
```

### PR
